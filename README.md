# ThreatLens AI
AI-Powered SOC Threat Detection and Alert Prioritization Platform

## Overview
ThreatLens AI is a Security Operations Center (SOC) dashboard that uses Machine Learning to detect cyber threats, classify network traffic, prioritize security alerts, and recommend mitigation actions.
The project demonstrates how AI can assist SOC analysts in reducing alert fatigue and improving incident response efficiency.

## Features
- Real-Time Threat Detection
- AI-Based Alert Prioritization
- Network Traffic Classification
- Security Alerts Dashboard
- Threat Risk Scoring
- Mitigation Recommendations
- Traffic Analyzer
- Model Performance Dashboard
- PCAP File Upload Analysis
- Random Forest Threat Classification

## Dataset
This project uses the NSL-KDD Intrusion Detection Dataset.
Attack Categories:
- Normal
- DoS (Denial of Service)
- Probe
- R2L (Remote to Local)
- U2R (User to Root)

## Unique Features
### AI Prioritized Threat Queue
Instead of displaying alerts randomly, ThreatLens AI ranks threats based on severity and risk score, helping analysts investigate the most critical incidents first.

### Explainable Threat Detection
The platform provides confidence scores and mitigation recommendations for detected threats.
